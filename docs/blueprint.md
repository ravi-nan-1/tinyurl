# **App Name**: LinkForge

## Core Features:

- URL Shortening: Shorten long URLs with a custom or randomly generated alias.
- Custom Alias: Generate a custom short link alias based on user input.
- Expiration Date: Set an expiration date for the shortened URL.
- Click Analytics: Track the number of clicks, device type, browser, country, referrer and timestamps for each shortened URL.
- Dashboard: View and manage all shortened URLs in a dashboard with options to copy, delete, and update aliases.
- Redirection API: Redirect to the target URL while logging analytics using a dedicated API endpoint.
- Spam Protection: Analyze user behavior and URL content as a tool for the system to determine if the new URL appears malicious, in order to protect against spam and abuse using rate limiting, and block malicious URLs.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to evoke trust and intelligence.
- Background color: Light lavender (#E6E6FA) to keep the design calm and focused.
- Accent color: Electric purple (#BF00FF) for interactive elements.
- Body and headline font: 'Inter', a sans-serif font, will give the website a neutral and objective feel.
- Use simple, clear icons for actions and categories, prioritizing comprehensibility.
- Maintain a clean, intuitive layout with a focus on ease of use and accessibility.
- Subtle transitions and animations for a smooth user experience.
